Today's Workers

Indian Standard Time (IST)

Users who completed at least one task today (IST)

—

Resets every day at 12:00 AM IST

ℹ️ How it's counted

A user is counted as a "worker" if they completed at least one of the following tasks today (IST):

  • 📅 Daily Check-in
  • ▶️ Watch Video Ad
  • 🧮 Math Quiz

The count resets automatically every day at 12:00 AM IST. A Telegram report is sent to admins at 11:59 PM IST each day.

Eligibility Rules

Set minimum daily task requirements for each action

Available Tasks: ▶️ Ad Watch (adWatch) 🧮 Math Quiz (mathQuiz) More tasks added here automatically when you create them.
💸

Withdrawal Eligibility

User must complete ALL listed tasks (minimum count each) to request withdrawal

If no tasks are added, withdrawal is always allowed (no daily requirement).

🎁

Coupon Claim Eligibility

User must complete ALL listed tasks (minimum count each) to claim a coupon

If no tasks are added, coupon claim is always allowed (no daily requirement).

📅

Daily Check-in Eligibility

User must complete ALL listed tasks (minimum count each) to claim daily check-in

If no tasks are added, check-in is always allowed (no daily requirement).
